What The Demo Shows

Solve the right-hand clue. Its answer sounds like something found in, on, or with the left-hand answer.

This page is a second puzzle format sitting alongside Chain Mind. It keeps the crossword-grid language, but swaps letter extraction for a sound connection.

The left-hand answers all belong to one group. Once you have enough of them, enter the connection at the bottom.

Example Logic

  • Peal sounds like peel, which points to orange.
  • Corps sounds like core, which points to apple.
  • Cedes sounds like seeds, which points to pomegranate.

Setter Notes

This is deliberately a gentle version: the sound bridge is shown after a correct clue answer. A harder version could hide that bridge and leave only the grid lengths.
